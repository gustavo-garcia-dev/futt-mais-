# ⚽ futt-mais+

> Plataforma para amantes de futebol acompanharem estatísticas, resultados e muito mais.

---

## **📌 Sobre o projeto**

O **futt-mais+** é um projeto desenvolvido com o objetivo de fornecer uma experiência completa para fãs de futebol, reunindo informações relevantes como:

- Historias dos times  
- Estatísticas de jogadores e times  
- Notícias e atualizações  
- Possível integração com APIs esportivas  

Este projeto foi criado para praticar desenvolvimento e explorar tecnologias modernas.

---

## **🚀 Tecnologias utilizadas**

Este projeto foi desenvolvido com as seguintes tecnologias:

- **Linguagens:** JavaScript / HTML / CSS  
- **Frameworks:** React / Node.js  
- **Banco de dados:** MySQL
